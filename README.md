# KafkaParty

Utility for reporting time-based partition lag metrics for Kafka consumer groups.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'kafka_party'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kafka_party
